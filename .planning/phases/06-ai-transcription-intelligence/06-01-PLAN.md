<objective>
Build the end-to-end transcription and AI analysis pipeline: when a call recording finishes downloading, the Go backend triggers the AI service, which splits the dual-channel audio, transcribes each channel, merges with speaker labels, runs Claude analysis to extract summary and CRM actions, and stores everything in a new call_transcripts table.

Purpose: This is the core intelligence layer that transforms raw call recordings into structured CRM data. Without it, recordings are just audio files. With it, every call automatically produces a transcript, summary, and actionable suggestions.

Output: Migration file, two new AI service modules (transcription.py, call_analysis.py), a new FastAPI route, and Go-side trigger integration.
</objective>

1. Create `backend/migrations/018_call_transcripts.sql`:
```sql
CREATE TABLE call_transcripts (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    call_id UUID NOT NULL REFERENCES call_logs(id) ON DELETE CASCADE,
    agent_id UUID NOT NULL REFERENCES users(id),
    full_text TEXT NOT NULL,
    speaker_segments JSONB NOT NULL DEFAULT '[]',
    ai_summary TEXT,
    ai_actions JSONB DEFAULT '[]',
    status TEXT NOT NULL DEFAULT 'pending',
    duration_seconds INTEGER,
    word_count INTEGER,
    created_at TIMESTAMPTZ DEFAULT NOW(),
    completed_at TIMESTAMPTZ
);

ALTER TABLE call_transcripts ENABLE ROW LEVEL SECURITY;
CREATE POLICY call_transcripts_agent ON call_transcripts
    USING (agent_id = current_setting('app.current_agent_id')::uuid);

CREATE UNIQUE INDEX idx_transcripts_call ON call_transcripts(call_id);
CREATE INDEX idx_transcripts_agent ON call_transcripts(agent_id, created_at DESC);
CREATE INDEX idx_transcripts_fulltext ON call_transcripts
    USING gin(to_tsvector('english', full_text));
```

3. Create `ai-service/app/services/transcription.py`:
- Import openai, pydub.AudioSegment, tempfile, os, logging
- Create module-level OpenAI client: `_client = openai.OpenAI(api_key=OPENAI_API_KEY) if OPENAI_API_KEY else None`
- Function `transcribe_audio_file(file_path: str) -> dict`: Opens file, calls `_client.audio.transcriptions.create(model="gpt-4o-transcribe", file=f, response_format="verbose_json", timestamp_granularities=["segment"])`, returns result dict.
- Function `split_channels_and_transcribe(local_path: str, direction: str) -> tuple[str, list[dict]]`:
  - Load stereo MP3 with `AudioSegment.from_mp3(local_path)`
  - Split to mono with `audio.split_to_mono()`
  - Channel mapping: if direction == "outbound", channel 0 = agent, channel 1 = client. If direction == "inbound", channel 0 = client, channel 1 = agent.
  - Export each channel to temp MP3 file using `tempfile.NamedTemporaryFile(suffix=".mp3", delete=False)`
  - Transcribe each channel with `transcribe_audio_file`
  - Build `speaker_segments` list: for each segment in agent transcript, add `{"speaker": "agent", "start": seg.start, "end": seg.end, "text": seg.text}`. Same for client with "client".
  - Sort all segments by `start` timestamp.
  - Build `full_text` by joining segments: `f"[{seg['speaker'].title()}]: {seg['text']}"` with newlines.
  - Clean up temp files in a `finally` block.
  - Return `(full_text, speaker_segments)`
  - Handle the case where audio has only 1 channel (mono): treat the single channel as the full conversation without speaker labels (speaker="unknown").

- Create module-level client: `_client = anthropic.Anthropic(api_key=ANTHROPIC_API_KEY)`
- Define `ANALYSIS_SYSTEM_PROMPT` as a constant string:
```
You are analyzing a real estate agent's phone call transcript.
Extract structured CRM updates from the conversation.
Return ONLY valid JSON with this exact structure:
{
  "summary": "2-3 sentence call summary",
  "tasks": [{"body": "task description", "due_date": "YYYY-MM-DD", "priority": "high|medium|low"}],
  "buyer_profile_updates": {"budget_max": null, "locations": [], "bedrooms": null, "bathrooms": null, "pre_approved": null, "timeline": null, "must_haves": [], "deal_breakers": [], "property_type": null},
  "deal_stage_suggestion": null,
  "key_quotes": ["quote1", "quote2"]
}
Only include fields where the transcript provides clear evidence. Use null for uncertain fields. For tasks, pick reasonable due dates relative to the call date. For deal_stage_suggestion, only suggest if there's clear evidence of stage progression (e.g., "Touring" if they discussed showings, "Offer" if they discussed making an offer).
```
- Function `analyze_transcript(transcript_text: str, contact_context: dict, call_metadata: dict) -> dict`:
  - Call `_client.messages.create(model=ANTHROPIC_MODEL, max_tokens=2048, system=ANALYSIS_SYSTEM_PROMPT, messages=[...])` with user message containing contact context, call metadata (direction, duration, date), and transcript text.
  - Parse response with `_parse_analysis_json(response.content[0].text)`.
  - Return parsed dict.
- Function `_parse_analysis_json(text: str) -> dict`:
  - Strip markdown code fences if present (regex: `r"```(?:json)?\s*([\s\S]*?)\s*```"`)
  - Try `json.loads(text)`.
  - If that fails, try extracting JSON block with regex `r"\{[\s\S]*\}"`.
  - If all fails, return `{"summary": text[:500], "tasks": [], "buyer_profile_updates": {}, "deal_stage_suggestion": None, "key_quotes": []}` as fallback.
- Function `build_ai_actions(analysis: dict, contact_id: str | None, call_metadata: dict) -> list[dict]`:
  - Convert analysis tasks into action dicts: `{"type": "create_task", "params": {"body": t["body"], "due_date": t.get("due_date"), "priority": t.get("priority", "medium"), "contact_id": contact_id}, "status": "pending"}`
  - If `analysis["buyer_profile_updates"]` has any non-null values and contact_id exists: add `{"type": "update_buyer_profile", "params": {"contact_id": contact_id, **non_null_updates}, "status": "pending"}`
  - If `analysis["deal_stage_suggestion"]` is not None and contact_id exists: add `{"type": "update_deal_stage", "params": {"contact_id": contact_id, "stage_name": analysis["deal_stage_suggestion"]}, "status": "pending"}`
  - Return the list.

    word_count = len(full_text.split())
  - Step 7: Update call_logs.transcription_status to 'completed'.
